<br/></br>### 课程介绍:<br/></br>数据可视化就是通过易读、易懂、易操作的图表，给用户带来良好的视觉效果，降低用户的理解难度，从而实现用数字给用户讲个故事的工作目的。
简单理解，数据可视化=数据+可视化，数据内容是基础，可视化是用图形化的方式呈现，并借此传达信息的方式。
所以数据内容是可视化的内核，单纯追求可视化炫酷的意义并不大，拥有了优质的数据内容，可视化的意义方得以凸显。
